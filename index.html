Connect Google Sheets

1 · Setup Script
2 · Deploy
3 · Connect

First, run the setup script to create all your Sheet tabs instantly. Open your Google Sheet, then:

  1. Extensions → Apps Script
  2. Delete any existing code — paste SoulSpark_Script.gs
  3. Click Save, then Run → runSetup
  4. Authorize when prompted — all tabs will be created

Deploy the Submission Script as a Web App so the form can send data:

  1. In Apps Script, make sure SoulSpark_Script.gs is open
  2. Click Deploy → New Deployment
  3. Type: Web App
  4. Execute as: Me
  5. Who has access: Anyone
  6. Click Deploy → Authorize → Copy the Web App URL
